eRisemail

  • A free e-mail server based on the Linux and MySQL/memcached.
  • Support SMTP, POP3, IMAP and WebMail
  • Support SSL/TLS
  • Support Web UI Management
  • Support iCalendar Format(RFC2445) / Mozilla Thunderbird Lightning Plugin
  • Support multiple authentication way(CRAM-MD5 DIGEST-MD5 APOP)
  • Support mail group and customizable policy
  • Support mail auditing
  • Support loading and extending Anti-junk Engine dynamically
  • Support the mail with huge size attachment( Maximum size is 2G )
  • Support mail rapid index and access
  • Recommended platform: Ubuntu and CentOS
  • Support running it in Cygwin at Windows
  • Project Wiki
