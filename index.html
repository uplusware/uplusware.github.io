eRisemail

  • Scalable distributed Mail/XMPP Server/Federation based on Linux/MySQL/Hadoop/LDAP.
  • Support SMTP, POP3, IMAP4, HTTP, XMPP and SSL/TLS.
  • Support Web Mail and Web Admin based on the built-in web server.
  • Support CRAM-MD5, DIGEST-MD5, APOP, EXTERNAL/TLS-Certification and GSSAPI/Kerberos.
  • Support Mail Group, Customize User Policy and Mail audit.
  • Support popular email client( Outlook, Thunderbird ...).
  • Support iCalendar and Mozilla Thunderbird Lightning Plugin.
  • Support LDAP Address Book based on built-in LDAP server.
  • Support the mail with huge size attachment( Maximum size is 2G ).
  • Support Apache SpamAssassin and the Self-Defined Anti-spam Engine.

UplusFTP

  • Lightweight green FTP server on Windows XP/7/8/10/server 2008/server 2012.
  • Support IPv4 and IPv6(EPRT/EPSV).
  • Support multi-users & multi-folders.
  • Support Web file accessing based on the built-in web server.
  • Support Windows Service mode.

Heaphttpd

  • Lightweight Web Server/Proxy/Tunneling/Reverse Proxy/HA Proxy with HTTP/1.1/2/S. (Installation)
  • Support HTTP Proxy, HTTPS Tunneling, HTTP Reverse/HA Proxy and HTTP Load Balancing.
  • Support extension, internal API, CGI/Fast-CGI(Spawn-fcgi)/SCGI, PHP/PHP-FPM, web.py and WebSocket.
  • Support Asynchronous I/O Mode and Distributed Deployment.
  • Support Service Object (For example, a long-live MariabDB or MongoDB connection).
