eRisemail

  • Free email server based on the Linux and MySQL/memcached
  • Support SMTP, POP3, IMAP, WebMail and SSL/TLS
  • Support backend administration on Web UI
  • Support iCalendar and Mozilla Thunderbird Lightning Plugin
  • Support multiple authentication methods(CRAM-MD5 DIGEST-MD5 APOP)
  • Support loading and extending Anti-junk Engine dynamically
  • Recommended platform: Ubuntu and CentOS
