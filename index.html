  • eRisemail is a scalable distributed Mail, IM and LDAP sever based on Linux and MySQL.
  • Support SMTP, POP3, IMAP4, HTTP and XMPP.
  • Support CRAM-MD5, DIGEST-MD5, APOP, EXTERNAL/TLS-Certification and GSSAPI/Kerberos.
  • Support Web Mail(support English and Simplified Chinese) and LDAP Address Book.
  • Support Mail Group, User Policy and External Mail Audit.
  • Support iCalendar and Mozilla Thunderbird Lightning Calendar.
  • Support the mail with huge size attachment of up to 2GB.
  • Support Apache SpamAssassin and the Self-Defined Anti-spam Engine.

  • Support HTTP/1.1, HTTP/2 and HTTPS.
  • Support HTTPS tunneling, Proxy, Reverse/HA Proxy, Cache and Load balancing function.
  • Support Extension, API module, CGI/FCGI/SCGI, PHP/PHP-FPM, web.py, WebSocket and Service Object.
  • Support highly concurrent asynchronous I/O mode and distributed deployment.
