  • eRisemail is a scalable distributed Mail, IM and LDAP sever based on Linux and MySQL(*The project began in 2004).
  • Support SMTP, POP3, IMAP4, HTTP and XMPP.
  • Support Web Mail and Web Admin based on a built-in web server.
  • Support CRAM-MD5, DIGEST-MD5, APOP, EXTERNAL/TLS-Certification and GSSAPI/Kerberos.
  • Support mail group, customizable user policy and mail audit.
  • Support iCalendar and Mozilla Thunderbird Lightning Plugin.
  • Support LDAP address book based on a built-in LDAP server.
  • Support the mail with huge size attachment of up to 2Gb.
  • Support Apache SpamAssassin and the Self-Defined Anti-spam Engine.

  • Heaphttpd is a lightweight web server on Linux.
  • Support HTTP/1.1, HTTP/2 and HTTPS.
  • Support HTTPS tunneling, proxy, reverse/HA proxy, cache and Load Balancing.
  • Support extension, dynamic-link module, CGI/FCGI/SCGI, PHP/PHP-FPM, web.py and WebSocket.
  • Support highly concurrent asynchronous I/O mode and distributed deployment.
  • Support Service Object (For example, a long-live MariabDB or MongoDB connection).
  • Wiki | Source Code
  • UplusFTP is a FTP server on Windows XP/2003/7/2008/8/2012/10(*The project began in 2004. Free but not open source).
  • Support IPv4 and IPv6(EPRT/EPSV).
  • Support multi-users & multi-folders.
  • Support Web file accessing based on the built-in web server.
  • Support Windows Service mode.
  • The lastest portable release: v2.0
  • For WinXP and Win2003, please use v1.7 (*Not support IPV6)
  • Bluewhale Gate is a lightweight asynchronous protocol independent proxy on Linux.
  • Support protocol independent reverse/HA proxy or load balancing.
  • Support highly concurrent asynchronous I/O.
  • Easy to scale the backend cluster.
